Basketball GM rosters for FIBA national teams.

From this page, you could find JSON roster files featuring FIBA national teams which you could use for Basketball GM.

To use the roster files, just do the following.

  • Visit the Basketball GM page.
  • Click New league >> Custom.
  • Under "Customize", select "Enter league file URL".
  • Use one of the URLs below in the URL field, and click Load.
  • Click "Create League" and enjoy!

Here are the list of roster file URLs:

  • https://raw.githubusercontent.com/leoisiah/fiba_bbgm/master/rosters/2020-21.FIBA.json
  • https://raw.githubusercontent.com/leoisiah/fiba_bbgm/master/rosters/2019-20.FIBA.json
  • https://raw.githubusercontent.com/leoisiah/fiba_bbgm/master/rosters/2018-19.FIBA.json
  • older files to follow, soon... (they already exist but I just want to refine them :D)
